# Privacy Policy

### Last updated: July 16, 2026

This application is designed to run entirely on the user's device and does not collect, store, transmit, or share any personal data.

## Data Collection

This application does not collect personal information, usage data, device identifiers, or any other information about its users.

The application does not use:
- User accounts
- Analytics services
- Advertising services
- Tracking technologies
- Cloud storage
- Remote servers

## Clipboard

When requested by the user, the application may copy a calculated result to the device's clipboard.

The application does not read or access any existing clipboard content.

## Sharing

The application allows users to share calculation results using the operating system's built-in sharing feature.

Sharing only occurs when explicitly initiated by the user. The application does not automatically send information to third parties.

## Data Storage

The application does not permanently store user data. Any values entered by the user exist only while the application is running and are discarded when it is closed.

## Changes to This Privacy Policy

If the application's data handling practices change in a future version, this Privacy Policy will be updated accordingly.

## Contact

If you have any questions about this Privacy Policy, please contact the developer using the contact information provided on the application's store page.
